SimSimi API Documentation

Interact with the SimSimi AI

Get Response

Use this endpoint to get a response from SimSimi.

GET https://simsimi-api-pro.onrender.com/sim?query=hi

Example Response:

{"respond":"hello"}

Teach SimSimi

Use this endpoint to teach SimSimi new responses.

GET https://simsimi-api-pro.onrender.com/teach?ask=hi&ans=pogi mo nmn hi pogi

Example Response:

{"respond":"SimSimi learned this new answer (359th) for question \"hi\" and the new response is \"pogi mo nmn hi pogi\""}
